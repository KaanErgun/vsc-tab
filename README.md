# VSC Tab

**Manage multiple projects in a single VS Code window with tabs.**

Stop juggling multiple VS Code windows! VSC Tab lets you organize and switch between projects instantly using a simple tab interface in the sidebar.

![Version](https://img.shields.io/visual-studio-marketplace/v/KaanErgun.vsc-tab?label=version) ![Installs](https://img.shields.io/visual-studio-marketplace/i/KaanErgun.vsc-tab) ![License](https://img.shields.io/github/license/KaanErgun/vsc-tab)

## Features

### Project Management
- **Add Projects** — Use the "+" button in the sidebar to register your projects
- **Remove Projects** — Clean up your project list anytime
- **Rename Projects** — Give custom names to your project tabs
- **Save Current Folder** — Quickly save your currently open folder as a tab

### Quick Switching
- **Single Click Switch** — Click a project tab to instantly open it
- **Status Bar** — See the currently active project in the status bar
- **Multi-Root Workspace** — Projects are added to your workspace, keeping everything accessible

### Flexible Opening
- **Open in Current Window** — Add projects to your current workspace
- **Open in New Window** — Launch a project in a separate VS Code window

## Usage

1. Click the **VSC Tab** icon in the Activity Bar (sidebar)
2. Click **"+"** to add a project folder
3. Click on any project to switch to it
4. Use the inline buttons to rename or remove projects

## Commands

| Command | Description |
|---------|-------------|
| `VSC Tab: Add Project` | Add a new project to the tab list |
| `VSC Tab: Save Current Folder as Tab` | Save the currently open folder as a tab |
| `VSC Tab: Switch to Project by ID` | Switch to a specific project |
| `Refresh` | Refresh the project list |
